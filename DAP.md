# 2. Objetivo

Desenvolver um website institucional responsivo para a DSJ Construtora, apresentando a empresa, seus diferenciais, obras realizadas e canais de contato. O projeto deve transmitir credibilidade, qualidade e profissionalismo por meio de uma interface moderna, intuitiva e alinhada à identidade visual da empresa.

---

# 3. Escopo

## O projeto contempla

- Desenvolvimento de website.
- HTML5 semântico.
- CSS3.
- JavaScript .
- Layout responsivo.
- Navegação em página única.
- Animações leves.
- Formulário de contato.
- Portfólio de obras.
- Links para redes sociais.

## O projeto não contempla

- Sistema de login.
- Área administrativa.
- Banco de dados.
- CMS.
- Blog.
- Área do cliente.
- Integrações externas além das definidas no projeto.
